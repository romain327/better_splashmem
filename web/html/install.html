Installation

Le code source du jeu est disponible sur en cliquant sur ce lien. Une fois le package .deb téléchargé, vous pouvez l'installer en ouvrant un terminal et en entrant les commandes suivantes :

  • cd $HOME/Téléchargements
  • sudo dpkg -i splashmem.deb
