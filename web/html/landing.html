Splashmem

Bienvenue sur le site web de splashmem, un petit jeu créé par les étudiants en 1ère année de cycle ingénieur en ISIE à Polytech Tours !

Vous trouverez de plus amples informations dans la section Comment jouer ?, vous trouverez les instructions d'installation du jeu dans la section Installation, et le code source du jeu est disponible sur GitHub !
