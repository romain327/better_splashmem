Règles du jeu

Lancement du jeu

Sur la page d'accueil, sélectionner les librairies avec lesquelles vous voulez jouer, puis cliquez sur créer. Ensuite, cliquez sur Télécharger. Une archive game.zip sera téléchargée. Une fois le téléchargement fini, allez dans vos téléchargements, et décompréssez l'archive. Ensuite, ouvrez le dossier décompressé et tapez la commande suivante :

  • chmod +x start.sh
  • ./start.sh

Liste des mouvements et coûts

  • Move up into Power up Tornado
  • ACTION_MOVE_x, x € {L, R, U, D}, Coût : 1 crédit ; se déplace dans la direction donnée
  • ACTION_DASH_x, x € {L, R, U, D}, Coût : 8 crédits ; se déplace de 8 cases dans la direction donnée en coloriant les 8 cases
  • ACTION_TELEPORT_x, x € {L, R, U, D}, Coût : 2 crédits ; se téléporte de 8 cases dans la direction donnée
  • ACTION_SPLASH, Coût : 9 crédits ; coliorie toutes les cases autour du joueur dans un rayon de 1
  • ACTION_BOMB, Coût : 9 crédits ; pose une bombe qui explose en 5 tours selon le même motif que ACTION_SPLASH
  • ACTION_ULTIME, Coût : tous les crédits restants

Un template de librairie est à télécharger ici.

Une fois votre librairie écrite, vous pouvez la compiler avec la commande suivante :

  • gcc -o lib.so -shared -fPIC lib.c
